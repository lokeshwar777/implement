Tailwind CSS configuration steps

  1. 1. Testing tailwindcss using the CDN, i.e by including a script tag in html
  2. 2. Tailwind CSS CLI setup
  3. npm init -y
  4. create an input.css file in src with -> @import "tailwindcss";
  5. take input from src/input.css and output it to build/css/output.css file
  6. configure run script in package.json -> "tailwind": "npx @tailwindcss/cli -i ./src/input.css -o ./build/css/output.css --watch"
  7. 3. prettier setup for automatic class sorting
  8. install the plugin as a dev dependency -> npm install -D prettier-plugin-tailwindcss
  9. add plugin to .prettierrc file -> { "plugins": ["prettier-plugin-tailwindcss"] }
  10. configure run script in package.json -> -> "format": "npx prettier --write **/*.html"
  11. 4. Vite setup
  12. You don't need to add the prettier script in the package.json

Use the commands in the terminal

  1. npm run tailwind -> see the live magic of tailwind on save
  2. npm run format -> to perform sorting of classes
